# 心理熵标签简表

这份简表用于统一项目中的“心理熵”标注口径，方便后续做数据清洗、人工复核和模型训练。

## 总体原则

- 心理熵不是临床诊断结果，而是当前心理状态的混乱度、失衡度与风险压力的综合估计。
- 标注时优先看“当前回合文本表达”，再结合历史消息和现实校园情境。
- 如果出现明显危机表达，风险优先级高于普通情绪困扰。

## 六个维度

### 1. 情绪强度

- 低：有轻度烦躁、担心，但仍可稳定表达
- 中：焦虑、难过、压抑明显，影响思路
- 高：绝望、崩溃、强烈痛苦占据主要表达

### 2. 情绪波动

- 低：情绪相对平稳
- 中：反复起伏，偶有失控感
- 高：明显冲动、惊恐、快速恶化或无法稳定下来

### 3. 认知负荷

- 低：问题清晰，能说出下一步
- 中：担忧多、任务堆积、思维打转
- 高：灾难化明显，脑中混乱、无法拆解问题

### 4. 生理失衡

- 低：没有明显睡眠或身体困扰
- 中：失眠、疲惫、食欲变化等开始影响状态
- 高：连续失眠、胸闷心慌、明显躯体化

### 5. 社会支持张力

- 低：有稳定支持关系
- 中：关系紧张、求助犹豫、支持不足
- 高：明显孤立、冲突升级、长期独自承受

### 6. 风险压力

- 低：无显著危险表达
- 中：有失控、绝望、撑不住等信号
- 高：有自残、自杀、伤人或明确危机表达

## 熵等级建议

- Level 1：`stable`
- Level 2：轻度 `strained`
- Level 3：中度 `strained`
- Level 4：`fragile`
- Level 5：`crisis`

## 减熵策略标注思路

- 先标最强驱动，不要求一次覆盖所有问题
- 优先选择能在 24-72 小时内执行的动作
- 方案需要同时满足“安全”“可执行”“校园可落地”
- 如果是危机内容，减熵第一目标不是安慰，而是现实安全转介
